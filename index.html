day22-better list view.

map view has to show every item and its details all at once.
Therefore it is very inappropriate for showing a large list.
It is not only waste of phone memory but also a lot of pressure to phone CPU.
However using list view, you can selectively show only what is on the visible range.
For this time,we will going to use FlatList.
We need to fill out three properties for FlatList component.

ListView Properties
  • data
    input data for flat list.
  • renderItem
    it determines how single data
    is going to be rendered
  • keyExtractor
    just like key element in
    map list, this also has key.
    ListView has some advantages over map view.
  • It has scrollView by default.
