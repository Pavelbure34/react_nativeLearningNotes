day23-action creator and library

    if you want to use actions from action creator in flatList,
    you gotta wire up actions into your application...
  1. Create pure javascript function that returns js object
    that would work as action creator and action respectively.
  2. Go to the component where you want to use actions
  3. Import the action creators into that component.
  4. Import connect helper function from react-redux library.
    connect function can also help wire actions into your application.
  5. wire the action to the component with connect function.
